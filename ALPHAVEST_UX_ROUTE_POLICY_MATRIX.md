## 6. Page-Type Policy Rules

| Page Type | Allowed | Forbidden | Default Codex Treatment |
|---|---|---|---|
| Hub | Orientation, status, queues, prioritized next work. | Full workflows, final destructive/release actions, deep tables. | Can receive UX-HUB tasks. |
| Workbench / Arbeitsseite | Active queue/table/review work, filters, selected context, action rail. | Unrelated metric card walls or deep object history. | Can receive UX-PAGE/UX-COMPLEXITY/UX-CTA tasks. |
| Detailseite | Object facts, evidence, audit/timeline, related objects and focused action. | Global route overload, unrelated queues. | Can receive D4 focused detail tasks. |
| Drawer Detail | Secondary context, snippets, evidence previews, audit snippets. | Complete workflow, release/export approval. | Context-only; must not become workflow. |
| Modal / Confirmation | Consequences, prerequisites, confirmation, validation, cancel/recovery. | New navigation or multi-page content. | Requires lifecycle and feedback tests. |
| Reference Page | Static reference/catalogue/state explanation. | Product CTAs and live workflow. | No product task. |
| P1 Deferred | Deferred placeholder/guard. | MVP action or primary nav elevation. | No MVP task. |
| Hold Blocked | Hold reason and blocked guard. | MVP CTA, implementation, visual generation. | No product task. |

## 7. Density Policy Rules

| Density Tier | Use | Policy |
|---|---|---|
| D1 Calm Executive | Client portal/mobile/executive summaries. | Use 3–5 cards, a clear released/hidden state, and one next-step panel. |
| D2 Productive Workbench | Analyst/advisor/compliance documents and review work. | Queue + selected context + gate/action rail; reduce decorative whitespace. |
| D3 Dense Operations | Governance, audit, RBAC, export scope/redaction and operational tables. | Dense tables with filters/sort/row actions; minimal decorative cards. |
| D4 Focused Detail | Decision, compliance review, advisor detail, evidence record and export preview/download. | Object header + key facts + evidence/audit sections + action rail. |
| P1/Hold/Reference density | Deferred/reference/blocked shells. | No MVP density work beyond safe guard and clarity. |
