# AlphaVest WealthOS Codex Handoff Pack

This package contains the context Codex needs to build the AlphaVest WealthOS browser click-dummy.

## Contents

- `AGENTS.md` — persistent Codex project instructions
- `CODEX_TASK_MASTER.md` — master implementation task
- `docs/ALPHAVEST_WEALTHOS_KB.md` — product knowledgebase
- `docs/WIREFRAME_MANIFEST.md` — reference image list
- `docs/SCREEN_SPECS.md` — route-by-route screen specs
- `docs/FUNCTIONAL_SCOPE.md` — required interactions and scope
- `docs/QUALITY_GATES.md` — acceptance and QA gates
- `public/reference/wireframes/` — 16 wireframe PNGs
- `public/reference/key-visuals/` — A0 / hero visual references

## How to Use

1. Create or open the target repository.
2. Copy all files from this pack into the repo root.
3. Start Codex with the contents of `CODEX_TASK_MASTER.md`.
4. Tell Codex: "Follow AGENTS.md. Read the KB and wireframes first. Implement the click-dummy."

## Why This Pack Exists

Codex cannot see the previous chat unless this context is placed into the repository or pasted into the task. This pack makes the product concept, designs, wireframes, functions and quality gates explicit.
