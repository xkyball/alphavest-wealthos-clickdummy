## Promotion Rule

A P1/P2 candidate can become a customer-facing clip only after:

- the audience question explicitly needs it,
- the current-reality label is acceptable for that audience,
- the caveat is safe in the caption or speaker note,
- proof path and run metadata are present,
- no route-static or metadata-only clip is described as a completed workflow.
