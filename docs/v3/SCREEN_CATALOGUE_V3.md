## Implementation interpretation

For each page:

1. Inspect the visual asset.
2. Inspect `PAGE_SPECS_V3.md` for data input/output and states.
3. Implement the route using shared components.
4. Use demo data from Prisma seed.
5. Implement modal/drawer states if the visual mode indicates them.
6. Keep UI clean; do not render specification metadata.
7. If the generated visual contains inconsistent spacing, normalize to the shared design system.
