# User Flow Definitions V3

## UF-01 Platform Admin configures platform
Actor: Platform Admin / Ops.
Pages: 007, 010, 011, 012.
Outcome: platform configuration, evidence templates and export controls exist.

## UF-02 Compliance Officer configures policies
Actor: Compliance Officer.
Pages: 008, 011, 012, 017.
Outcome: advice boundary, evidence requirements and export/redaction rules are active.

## UF-03 New user accepts invitation
Actor: Invited user.
Pages: 003, 004, 005, 006, then 001/002 for future entry.
Outcome: user profile, consent record and role assignment.

## UF-04 Principal reviews dashboard and submits profile
Actor: Principal / Family CFO.
Pages: 019, 021, 022, 023.
Outcome: family profile and relationship records ready for review.

## UF-05 Family CFO / Analyst creates structure
Actor: Principal / CFO / Analyst / Advisor.
Pages: 024, 025, 026, 031.
Outcome: entity graph, ownership participants and structure gaps.

## UF-06 Client uploads document
Actor: Client / CFO / External Advisor.
Pages: 027, 028, 029, 030.
Outcome: document record, extraction draft, analyst review task.

## UF-07 Analyst processes signal
Actor: Analyst.
Pages: 033, 034, 035, 032.
Outcome: trigger classified, action item created, draft route selected.

## UF-08 Advisor reviews recommendation
Actor: Senior Wealth Advisor.
Pages: 036, 037.
Outcome: approve/revise/request data/escalate. Client visibility remains blocked.

## UF-09 Compliance releases or blocks
Actor: Compliance Officer.
Pages: 038, 039, 040, 041, 042.
Outcome: release to client or block/request evidence with audit trail.

## UF-10 Client decides
Actor: Principal / Family Council / Trustee scoped.
Pages: 043, 044, 045.
Outcome: decision accepted/deferred/rejected, evidence package created.

## UF-11 Evidence review and export
Actor: Client / Advisor / Compliance / Privacy scoped.
Pages: 046, 047, 051, 054, 055, 056, 057, 058.
Outcome: evidence viewed/exported with redaction and audit.

## UF-12 Governance access change
Actor: Principal / Admin / Compliance.
Pages: 048, 049, 050, 051.
Outcome: role assignment, access request decision, audit event.

## UF-13 Communication path selection
Actor: Advisor / Client Success.
Pages: 052, 053.
Outcome: secure message, call, F2F or external specialist path.

## UF-14 Operations monitors workload and SLA
Actor: Ops / Programme / Leads.
Pages: 059, 060, 061, 062, 063.
Outcome: bottlenecks, SLA breaches and implementation state visible.
